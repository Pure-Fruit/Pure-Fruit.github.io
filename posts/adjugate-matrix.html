伴随矩阵的性质与证明

Properties and Proofs of Adjugate Matrix

2026-01-17 • Pure Tao

数学公式为通用符号，中英文共用

Mathematical formulas use universal notation

定义

设 \(A = (a_{ij}) \in F^{n \times n}\) 为 \(n \times n\) 方阵。记：

  • \(M_{ij}\)：删去第 \(i\) 行第 \(j\) 列得到的 \((n-1) \times (n-1)\) 子式（minor）
  • \(C_{ij} = (-1)^{i+j} M_{ij}\)：代数余子式（cofactor）

伴随矩阵（adjugate）定义为：

\[ A^* = \text{adj}(A) = (C_{ij})^T \]

即 \((A^*)_{ij} = C_{ji}\)。

1. 核心恒等式

命题：

\[ AA^* = A^*A = |A|I \]

证明（以 \(AA^*\) 为例）：

看矩阵乘积 \(AA^*\) 的第 \((i,k)\) 个元素：

\[ (AA^*)_{ik} = \sum_{j=1}^{n} a_{ij}(A^*)_{jk} = \sum_{j=1}^{n} a_{ij} C_{kj} \]

当 \(i = k\) 时：这是行列式对第 \(k\) 行做拉普拉斯展开：

\[ \sum_{j=1}^{n} a_{kj} C_{kj} = |A| \]

当 \(i \neq k\) 时：构造矩阵 \(B\)：把 \(A\) 的第 \(k\) 行替换成第 \(i\) 行，其余行不变。则 \(B\) 有两行相同，所以 \(|B| = 0\)。另一方面，\(B\) 的行列式对第 \(k\) 行展开：

\[ |B| = \sum_{j=1}^{n} b_{kj} C_{kj} = \sum_{j=1}^{n} a_{ij} C_{kj} = 0 \]

综上：

\[ (AA^*)_{ik} = \begin{cases} |A|, & i = k \\ 0, & i \neq k \end{cases} \]

即 \(AA^* = |A|I\)。同理可证 \(A^*A = |A|I\)。∎

2. 与逆矩阵的关系

命题：若 \(|A| \neq 0\)（即 \(A\) 可逆），则：

\[ A^{-1} = \frac{1}{|A|} A^* \]

证明：由核心恒等式 \(AA^* = |A|I\)，两边同除以 \(|A| \neq 0\)：

\[ A \left( \frac{1}{|A|} A^* \right) = I \]

因此 \(\frac{1}{|A|} A^*\) 是 \(A\) 的右逆。方阵右逆必等于逆矩阵，故结论成立。∎

3. 行列式性质

命题：

\[ |A^*| = |A|^{n-1} \]

证明：

(1) 当 \(|A| \neq 0\) 时：由 \(A^* = |A| A^{-1}\)，取行列式：

\[ |A^*| = ||A| A^{-1}| = |A|^n \cdot |A^{-1}| = |A|^n \cdot \frac{1}{|A|} = |A|^{n-1} \]

(2) 当 \(|A| = 0\) 时：右边为 \(0^{n-1} = 0\)（当 \(n \geq 2\)）。此时由核心恒等式 \(AA^* = 0\)。若 \(A^*\) 可逆，则左乘 \(A^{*-1}\) 得 \(A = 0\)，但 \(A = 0\) 的伴随矩阵也为 0，不可逆，矛盾；因此 \(A^*\) 不可逆，\(|A^*| = 0\)。∎

4. 数乘性质

命题：对标量 \(k\)：

\[ (kA)^* = k^{n-1} A^* \]

证明：\((kA)\) 的每个 \((n-1) \times (n-1)\) 子式都把 \(A\) 中的元素整体乘了 \(k\)，一共乘了 \(n-1\) 个元素，所以：

\[ M_{ij}(kA) = k^{n-1} M_{ij}(A) \]

代数余子式仅多了符号 \((-1)^{i+j}\)，不影响倍数，因此：

\[ C_{ij}(kA) = k^{n-1} C_{ij}(A) \]

再转置即得 \((kA)^* = k^{n-1} A^*\)。∎

5. 乘积性质

命题：对任意 \(n \times n\) 方阵 \(A, B\)：

\[ (AB)^* = B^* A^* \]

次序反转，与逆矩阵类似。

证明：对矩阵 \(AB\) 使用核心恒等式：

\[ (AB)(AB)^* = |AB|I = |A||B|I \]

另一方面：

\[ (AB)(B^*A^*) = A(BB^*)A^* = A(|B|I)A^* = |B|(AA^*) = |B|(|A|I) = |A||B|I \]

所以 \((AB)^*\) 与 \(B^*A^*\) 都满足同一恒等式。由伴随矩阵定义的唯一性，\((AB)^* = B^*A^*\)。∎

6. 转置性质

命题：

\[ (A^T)^* = (A^*)^T \]

证明：\(A^T\) 的 \((i,j)\) 余子式对应于 \(A\) 的 \((j,i)\) 余子式（删行删列在转置下互换），并且代数余子式的符号 \((-1)^{i+j}\) 不变，因此：

\[ C_{ij}(A^T) = C_{ji}(A) \]

于是：

\[ (A^T)^* = (C_{ij}(A^T))^T = (C_{ji}(A))^T = (A^*)^T \]

∎

7. 伴随的伴随

命题：

\[ (A^*)^* = |A|^{n-2} A \]

特别：若 \(|A| = 1\)，则 \((A^*)^* = A\)。

证明：若 \(|A| \neq 0\)，由 \(A^* = |A| A^{-1}\)，得：

\[ (A^*)^{-1} = \frac{1}{|A|} A \]

再用"伴随与逆矩阵"关系对 \(A^*\)：

\[ (A^*)^* = |A^*| (A^*)^{-1} = |A|^{n-1} \cdot \frac{1}{|A|} A = |A|^{n-2} A \]

当 \(|A| = 0\) 时，右侧为 0（若 \(n \geq 3\)），两边作为多项式函数在稠密集合上相等，因此对所有 \(A\) 成立。∎

8. 不可逆时的秩结论

若 \(|A| = 0\)，则 \(AA^* = A^*A = 0\)，并且有如下结论：

  • 若 \(\text{rank}(A) = n-1\)，则 \(\text{rank}(A^*) = 1\)
  • 若 \(\text{rank}(A) \leq n-2\)，则 \(A^* = 0\)

证明要点：

\(\text{rank}(A) \leq n-2\) 意味着所有 \((n-1)\) 阶子式全为 0，因此所有代数余子式全为 0，从而 \(A^* = 0\)。

\(\text{rank}(A) = n-1\) 意味着至少存在一个 \((n-1)\) 阶子式非零，所以 \(A^* \neq 0\)。同时由 \(|A| = 0\) 得 \(AA^* = 0\)，于是 \(A^*\) 的每一列都在 \(A\) 的零空间中。此时 \(\dim \ker(A) = 1\)，所以 \(A^*\) 的所有列都共线，\(\text{rank}(A^*) \leq 1\)。再结合 \(A^* \neq 0\)，得 \(\text{rank}(A^*) = 1\)。∎

9. 行操作的影响

  • 交换两行：伴随矩阵乘以 \(-1\)
  • 某行乘以 \(k\)：伴随矩阵乘以 \(k\)
  • 一行加另一行的倍数：伴随矩阵不变

这一点与代数余子式的构成一致。

10. 特征值关系（可逆时）

若 \(A\) 可逆且特征值为 \(\lambda_i\)，则：

\[ A^* = |A| A^{-1} \]

所以其特征值为：

\[ \mu_i = |A| \lambda_i^{-1} \]

简要总结

  • 伴随矩阵是余子式矩阵的转置
  • 满足 \(AA^* = A^*A = |A|I\)
  • 可逆矩阵满足 \(A^{-1} = \frac{A^*}{|A|}\)
  • \(|A^*| = |A|^{n-1}\) 且 \((A^*)^* = |A|^{n-2}A\)
  • 不可逆时，伴随矩阵的秩与 \(A\) 的秩紧密相关

Definition

Let \(A = (a_{ij}) \in F^{n \times n}\) be an \(n \times n\) square matrix. Define:

  • \(M_{ij}\): the \((n-1) \times (n-1)\) minor obtained by deleting row \(i\) and column \(j\)
  • \(C_{ij} = (-1)^{i+j} M_{ij}\): the cofactor

The adjugate matrix is defined as:

\[ A^* = \text{adj}(A) = (C_{ij})^T \]

That is, \((A^*)_{ij} = C_{ji}\).

1. Core Identity

Proposition:

\[ AA^* = A^*A = |A|I \]

Proof (taking \(AA^*\) as example):

Consider the \((i,k)\)-th entry of the matrix product \(AA^*\):

\[ (AA^*)_{ik} = \sum_{j=1}^{n} a_{ij}(A^*)_{jk} = \sum_{j=1}^{n} a_{ij} C_{kj} \]

When \(i = k\): This is the Laplace expansion of the determinant along row \(k\):

\[ \sum_{j=1}^{n} a_{kj} C_{kj} = |A| \]

When \(i \neq k\): Construct matrix \(B\) by replacing row \(k\) of \(A\) with row \(i\), leaving other rows unchanged. Then \(B\) has two identical rows, so \(|B| = 0\). On the other hand, expanding the determinant of \(B\) along row \(k\):

\[ |B| = \sum_{j=1}^{n} b_{kj} C_{kj} = \sum_{j=1}^{n} a_{ij} C_{kj} = 0 \]

In summary:

\[ (AA^*)_{ik} = \begin{cases} |A|, & i = k \\ 0, & i \neq k \end{cases} \]

Thus \(AA^* = |A|I\). Similarly, \(A^*A = |A|I\). ∎

2. Relation to Inverse Matrix

Proposition: If \(|A| \neq 0\) (i.e., \(A\) is invertible), then:

\[ A^{-1} = \frac{1}{|A|} A^* \]

Proof: From the core identity \(AA^* = |A|I\), dividing both sides by \(|A| \neq 0\):

\[ A \left( \frac{1}{|A|} A^* \right) = I \]

Therefore \(\frac{1}{|A|} A^*\) is the right inverse of \(A\). For square matrices, the right inverse equals the inverse, so the conclusion holds. ∎

3. Determinant Property

Proposition:

\[ |A^*| = |A|^{n-1} \]

Proof:

(1) When \(|A| \neq 0\): From \(A^* = |A| A^{-1}\), taking determinant:

\[ |A^*| = ||A| A^{-1}| = |A|^n \cdot |A^{-1}| = |A|^n \cdot \frac{1}{|A|} = |A|^{n-1} \]

(2) When \(|A| = 0\): The right side is \(0^{n-1} = 0\) (when \(n \geq 2\)). From the core identity \(AA^* = 0\). If \(A^*\) were invertible, multiplying on the left by \(A^{*-1}\) gives \(A = 0\), but the adjugate of \(A = 0\) is also 0, which is not invertible, a contradiction; therefore \(A^*\) is not invertible, \(|A^*| = 0\). ∎

4. Scalar Multiplication Property

Proposition: For scalar \(k\):

\[ (kA)^* = k^{n-1} A^* \]

Proof: Each \((n-1) \times (n-1)\) minor of \((kA)\) multiplies the elements of \(A\) by \(k\), with a total of \(n-1\) elements multiplied, so:

\[ M_{ij}(kA) = k^{n-1} M_{ij}(A) \]

The cofactor only adds the sign \((-1)^{i+j}\), which doesn't affect the multiplier, therefore:

\[ C_{ij}(kA) = k^{n-1} C_{ij}(A) \]

Transposing gives \((kA)^* = k^{n-1} A^*\). ∎

5. Product Property

Proposition: For any \(n \times n\) square matrices \(A, B\):

\[ (AB)^* = B^* A^* \]

The order is reversed, similar to inverse matrices.

Proof: Applying the core identity to matrix \(AB\):

\[ (AB)(AB)^* = |AB|I = |A||B|I \]

On the other hand:

\[ (AB)(B^*A^*) = A(BB^*)A^* = A(|B|I)A^* = |B|(AA^*) = |B|(|A|I) = |A||B|I \]

So both \((AB)^*\) and \(B^*A^*\) satisfy the same identity. By the uniqueness of the adjugate matrix definition, \((AB)^* = B^*A^*\). ∎

6. Transpose Property

Proposition:

\[ (A^T)^* = (A^*)^T \]

Proof: The \((i,j)\) cofactor of \(A^T\) corresponds to the \((j,i)\) cofactor of \(A\) (row and column deletion swap under transpose), and the sign \((-1)^{i+j}\) of the cofactor remains unchanged, therefore:

\[ C_{ij}(A^T) = C_{ji}(A) \]

Thus:

\[ (A^T)^* = (C_{ij}(A^T))^T = (C_{ji}(A))^T = (A^*)^T \]

∎

7. Adjugate of Adjugate

Proposition:

\[ (A^*)^* = |A|^{n-2} A \]

In particular: if \(|A| = 1\), then \((A^*)^* = A\).

Proof: If \(|A| \neq 0\), from \(A^* = |A| A^{-1}\), we get:

\[ (A^*)^{-1} = \frac{1}{|A|} A \]

Using the "adjugate and inverse" relation for \(A^*\):

\[ (A^*)^* = |A^*| (A^*)^{-1} = |A|^{n-1} \cdot \frac{1}{|A|} A = |A|^{n-2} A \]

When \(|A| = 0\), the right side is 0 (if \(n \geq 3\)), and both sides are equal as polynomial functions on a dense set, so the result holds for all \(A\). ∎

8. Rank Conclusions for Singular Matrices

If \(|A| = 0\), then \(AA^* = A^*A = 0\), and we have the following conclusions:

  • If \(\text{rank}(A) = n-1\), then \(\text{rank}(A^*) = 1\)
  • If \(\text{rank}(A) \leq n-2\), then \(A^* = 0\)

Key points of proof:

\(\text{rank}(A) \leq n-2\) means all \((n-1)\)-order minors are 0, so all cofactors are 0, thus \(A^* = 0\).

\(\text{rank}(A) = n-1\) means at least one \((n-1)\)-order minor is nonzero, so \(A^* \neq 0\). Also from \(|A| = 0\) we have \(AA^* = 0\), so every column of \(A^*\) lies in the null space of \(A\). Since \(\dim \ker(A) = 1\), all columns of \(A^*\) are collinear, \(\text{rank}(A^*) \leq 1\). Combined with \(A^* \neq 0\), we get \(\text{rank}(A^*) = 1\). ∎

9. Effect of Row Operations

  • Swapping two rows: adjugate matrix is multiplied by \(-1\)
  • Multiplying a row by \(k\): adjugate matrix is multiplied by \(k\)
  • Adding a multiple of one row to another: adjugate matrix unchanged

This is consistent with how cofactors are constructed.

10. Eigenvalue Relation (Invertible Case)

If \(A\) is invertible with eigenvalues \(\lambda_i\), then:

\[ A^* = |A| A^{-1} \]

So its eigenvalues are:

\[ \mu_i = |A| \lambda_i^{-1} \]

Summary

  • The adjugate matrix is the transpose of the cofactor matrix
  • Satisfies \(AA^* = A^*A = |A|I\)
  • For invertible matrices: \(A^{-1} = \frac{A^*}{|A|}\)
  • \(|A^*| = |A|^{n-1}\) and \((A^*)^* = |A|^{n-2}A\)
  • For singular matrices, the rank of the adjugate is closely related to the rank of \(A\)
