伴随矩阵的性质与证明

2026-01-17 • Pure Tao

定义

设 \(A = (a_{ij}) \in F^{n \times n}\) 为 \(n \times n\) 方阵。记：

  • \(M_{ij}\)：删去第 \(i\) 行第 \(j\) 列得到的 \((n-1) \times (n-1)\) 子式（minor）
  • \(C_{ij} = (-1)^{i+j} M_{ij}\)：代数余子式（cofactor）

伴随矩阵（adjugate）定义为：

\[ A^* = \text{adj}(A) = (C_{ij})^T \]

即 \((A^*)_{ij} = C_{ji}\)。

1. 核心恒等式

命题：

\[ AA^* = A^*A = |A|I \]

证明（以 \(AA^*\) 为例）：

看矩阵乘积 \(AA^*\) 的第 \((i,k)\) 个元素：

\[ (AA^*)_{ik} = \sum_{j=1}^{n} a_{ij}(A^*)_{jk} = \sum_{j=1}^{n} a_{ij} C_{kj} \]

当 \(i = k\) 时：这是行列式对第 \(k\) 行做拉普拉斯展开：

\[ \sum_{j=1}^{n} a_{kj} C_{kj} = |A| \]

当 \(i \neq k\) 时：构造矩阵 \(B\)：把 \(A\) 的第 \(k\) 行替换成第 \(i\) 行，其余行不变。则 \(B\) 有两行相同，所以 \(|B| = 0\)。另一方面，\(B\) 的行列式对第 \(k\) 行展开：

\[ |B| = \sum_{j=1}^{n} b_{kj} C_{kj} = \sum_{j=1}^{n} a_{ij} C_{kj} = 0 \]

综上：

\[ (AA^*)_{ik} = \begin{cases} |A|, & i = k \\ 0, & i \neq k \end{cases} \]

即 \(AA^* = |A|I\)。同理可证 \(A^*A = |A|I\)。∎

2. 与逆矩阵的关系

命题：若 \(|A| \neq 0\)（即 \(A\) 可逆），则：

\[ A^{-1} = \frac{1}{|A|} A^* \]

证明：由核心恒等式 \(AA^* = |A|I\)，两边同除以 \(|A| \neq 0\)：

\[ A \left( \frac{1}{|A|} A^* \right) = I \]

因此 \(\frac{1}{|A|} A^*\) 是 \(A\) 的右逆。方阵右逆必等于逆矩阵，故结论成立。∎

3. 行列式性质

命题：

\[ |A^*| = |A|^{n-1} \]

证明：

(1) 当 \(|A| \neq 0\) 时：由 \(A^* = |A| A^{-1}\)，取行列式：

\[ |A^*| = ||A| A^{-1}| = |A|^n \cdot |A^{-1}| = |A|^n \cdot \frac{1}{|A|} = |A|^{n-1} \]

(2) 当 \(|A| = 0\) 时：右边为 \(0^{n-1} = 0\)（当 \(n \geq 2\)）。此时由核心恒等式 \(AA^* = 0\)。若 \(A^*\) 可逆，则左乘 \(A^{*-1}\) 得 \(A = 0\)，但 \(A = 0\) 的伴随矩阵也为 0，不可逆，矛盾；因此 \(A^*\) 不可逆，\(|A^*| = 0\)。∎

4. 数乘性质

命题：对标量 \(k\)：

\[ (kA)^* = k^{n-1} A^* \]

证明：\((kA)\) 的每个 \((n-1) \times (n-1)\) 子式都把 \(A\) 中的元素整体乘了 \(k\)，一共乘了 \(n-1\) 个元素，所以：

\[ M_{ij}(kA) = k^{n-1} M_{ij}(A) \]

代数余子式仅多了符号 \((-1)^{i+j}\)，不影响倍数，因此：

\[ C_{ij}(kA) = k^{n-1} C_{ij}(A) \]

再转置即得 \((kA)^* = k^{n-1} A^*\)。∎

5. 乘积性质

命题：对任意 \(n \times n\) 方阵 \(A, B\)：

\[ (AB)^* = B^* A^* \]

次序反转，与逆矩阵类似。

证明：对矩阵 \(AB\) 使用核心恒等式：

\[ (AB)(AB)^* = |AB|I = |A||B|I \]

另一方面：

\[ (AB)(B^*A^*) = A(BB^*)A^* = A(|B|I)A^* = |B|(AA^*) = |B|(|A|I) = |A||B|I \]

所以 \((AB)^*\) 与 \(B^*A^*\) 都满足同一恒等式。由伴随矩阵定义的唯一性，\((AB)^* = B^*A^*\)。∎

6. 转置性质

命题：

\[ (A^T)^* = (A^*)^T \]

证明：\(A^T\) 的 \((i,j)\) 余子式对应于 \(A\) 的 \((j,i)\) 余子式（删行删列在转置下互换），并且代数余子式的符号 \((-1)^{i+j}\) 不变，因此：

\[ C_{ij}(A^T) = C_{ji}(A) \]

于是：

\[ (A^T)^* = (C_{ij}(A^T))^T = (C_{ji}(A))^T = (A^*)^T \]

∎

7. 伴随的伴随

命题：

\[ (A^*)^* = |A|^{n-2} A \]

特别：若 \(|A| = 1\)，则 \((A^*)^* = A\)。

证明：若 \(|A| \neq 0\)，由 \(A^* = |A| A^{-1}\)，得：

\[ (A^*)^{-1} = \frac{1}{|A|} A \]

再用"伴随与逆矩阵"关系对 \(A^*\)：

\[ (A^*)^* = |A^*| (A^*)^{-1} = |A|^{n-1} \cdot \frac{1}{|A|} A = |A|^{n-2} A \]

当 \(|A| = 0\) 时，右侧为 0（若 \(n \geq 3\)），两边作为多项式函数在稠密集合上相等，因此对所有 \(A\) 成立。∎

8. 不可逆时的秩结论

若 \(|A| = 0\)，则 \(AA^* = A^*A = 0\)，并且有如下结论：

  • 若 \(\text{rank}(A) = n-1\)，则 \(\text{rank}(A^*) = 1\)
  • 若 \(\text{rank}(A) \leq n-2\)，则 \(A^* = 0\)

证明要点：

\(\text{rank}(A) \leq n-2\) 意味着所有 \((n-1)\) 阶子式全为 0，因此所有代数余子式全为 0，从而 \(A^* = 0\)。

\(\text{rank}(A) = n-1\) 意味着至少存在一个 \((n-1)\) 阶子式非零，所以 \(A^* \neq 0\)。同时由 \(|A| = 0\) 得 \(AA^* = 0\)，于是 \(A^*\) 的每一列都在 \(A\) 的零空间中。此时 \(\dim \ker(A) = 1\)，所以 \(A^*\) 的所有列都共线，\(\text{rank}(A^*) \leq 1\)。再结合 \(A^* \neq 0\)，得 \(\text{rank}(A^*) = 1\)。∎

9. 行操作的影响

  • 交换两行：伴随矩阵乘以 \(-1\)
  • 某行乘以 \(k\)：伴随矩阵乘以 \(k\)
  • 一行加另一行的倍数：伴随矩阵不变

这一点与代数余子式的构成一致。

10. 特征值关系（可逆时）

若 \(A\) 可逆且特征值为 \(\lambda_i\)，则：

\[ A^* = |A| A^{-1} \]

所以其特征值为：

\[ \mu_i = |A| \lambda_i^{-1} \]

简要总结

  • 伴随矩阵是余子式矩阵的转置
  • 满足 \(AA^* = A^*A = |A|I\)
  • 可逆矩阵满足 \(A^{-1} = \frac{A^*}{|A|}\)
  • \(|A^*| = |A|^{n-1}\) 且 \((A^*)^* = |A|^{n-2}A\)
  • 不可逆时，伴随矩阵的秩与 \(A\) 的秩紧密相关
