Swin Transformer 图像分类

Swin Transformer Image Classification

查看源码View Source • PyTorch

项目简介

本项目使用 PyTorch 实现了 Swin Transformer 图像分类任务。Swin Transformer 是 ICCV 2021 Best Paper，一种层次化的 Vision Transformer，通过移位窗口机制实现高效的自注意力计算。

主要特性

  • 完整的 Swin Transformer 实现（Tiny/Small/Base/Large 变体）
  • 支持 ImageNet 预训练权重迁移学习
  • 训练、验证、预测完整流程
  • 混淆矩阵可视化
  • TensorBoard 日志记录
  • 错误样本分析工具

项目结构

swin_transformer/
├── model.py                    # 模型架构
├── train.py                    # 训练脚本
├── predict.py                  # 单张图片预测
├── my_dataset.py               # 自定义数据集类
├── utils.py                    # 工具函数
├── create_confusion_matrix.py  # 混淆矩阵可视化
├── select_incorrect_samples.py # 错误样本分析
└── weights/                    # 模型权重

快速开始

安装依赖：

pip install torch torchvision tqdm matplotlib prettytable tensorboard pillow

训练模型：

python train.py --data-path ./flower_photos --weights ./swin_tiny_patch4_window7_224.pth --epochs 10

训练效果

在花卉数据集（5类）上使用 Swin-Tiny 训练，10 轮后验证集准确率可达 95%+。

Project Overview

This project implements Swin Transformer for image classification using PyTorch. Swin Transformer is the ICCV 2021 Best Paper, a hierarchical Vision Transformer that achieves efficient self-attention computation through shifted window mechanism.

Key Features

  • Complete Swin Transformer implementation (Tiny/Small/Base/Large variants)
  • Support for ImageNet pretrained weights transfer learning
  • Complete training, validation, and prediction pipeline
  • Confusion matrix visualization
  • TensorBoard logging
  • Error sample analysis tools

Project Structure

swin_transformer/
├── model.py                    # Model architecture
├── train.py                    # Training script
├── predict.py                  # Single image prediction
├── my_dataset.py               # Custom dataset class
├── utils.py                    # Utility functions
├── create_confusion_matrix.py  # Confusion matrix visualization
├── select_incorrect_samples.py # Error sample analysis
└── weights/                    # Model weights

Quick Start

Install dependencies:

pip install torch torchvision tqdm matplotlib prettytable tensorboard pillow

Train model:

python train.py --data-path ./flower_photos --weights ./swin_tiny_patch4_window7_224.pth --epochs 10

Training Results

Using Swin-Tiny on flower dataset (5 classes), validation accuracy reaches 95%+ after 10 epochs.
