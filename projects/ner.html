BERT-NER 中文命名实体识别

查看源码 • PyTorch

项目简介

本项目使用 BERT 在 CLUENER 数据集上实现中文命名实体识别（NER）。NER 是一种序列标注任务，用于识别文本中的命名实体（人名、机构名、地名等）。

主要特性

  • 基于 BERT 的中文 NER 模型
  • 支持 CLUENER 数据集的 10 种实体类型
  • BIO 标注体系实现
  • 简单易用的预测 API
  • 使用 seqeval 进行全面评估

支持的实体类型

  • 人名 (name)：张三、李白
  • 地址 (address)：北京市海淀区
  • 公司 (company)：阿里巴巴、腾讯
  • 政府机构 (government)：国务院、教育部
  • 组织 (organization)：清华大学、联合国
  • 书籍 (book)：红楼梦、三国演义
  • 电影 (movie)：流浪地球
  • 游戏 (game)：王者荣耀
  • 职位 (position)：总经理、教授
  • 景点 (scene)：故宫、长城

快速开始

安装依赖：

pip install -r requirements.txt

下载数据和模型：

python download_data.py

训练模型：

python bert_ner.py

预测：

python predict.py

使用示例

from predict import NERPredictor

# 初始化预测器
predictor = NERPredictor('./bert_ner_model')

# 预测实体
text = "张三在北京大学读书"
entities = predictor.predict(text)

for entity in entities:
    print(f"{entity['text']} - {entity['type']}")
# 输出：
# 张三 - name
# 北京大学 - organization

BIO 标注体系

文本：张 三 在 北 京 大 学 读 书
标签：B-name I-name O B-organization I-organization I-organization I-organization O O
  • B-XXX：实体开始
  • I-XXX：实体内部
  • O：非实体
